# CS 261: ML at Scale Project

## Project Overview
This repository contains the materials for our CS 261 (ML at Scale) project. The project focuses on building a scalable machine learning pipeline to analyze and predict delays based on various factors, including weather conditions.

## Project Structure
The project is organized into several key directories containing visuals and data supporting our ML pipeline and analysis:

* **`data_photos/`**: Visualizations from our Exploratory Data Analysis (EDA).
* **`pm_photos/`**: Project management and architectural diagrams.
* **`profile_photos/`**: Project team members.

## Machine Learning Pipeline
Our project implements a full ML pipeline designed for scale, covering:
1. Data Ingestion & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering (including weather-integrated features)
4. Model Training & Validation
5. Scaling & Performance Optimization

## Analysis Insights
The exploratory analysis focuses on the relationship between weather patterns and target delays, as documented in the `data_photos` directory.
